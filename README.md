# etl_final_homework

В рамках выполнения задач была проведена работа по организации ETL-процессов с использованием облачных сервисов Yandex Cloud. Основные этапы включали:

Перенос данных: Создана база данных YDB, в которую загружены подготовленные CSV-файлы объёмом более 30 Мб. Настроен и выполнен трансфер данных из YDB в Object Storage с помощью сервиса Data Transfer.

Оркестрация и обработка: Развёрнута инфраструктура для Apache Airflow, написан DAG для автоматического создания кластера Data Processing, запуска PySpark-задания (обработка и партиционирование данных) и последующего удаления кластера. Данные прочитаны из Object Storage, преобразованы и записаны обратно.

Визуализация: В Yandex DataLens создано подключение к YDB, на основе данных построен датасет и разработан дашборд с графиками

Все выполненные задания подкреплены скриншотами
